
# Practice 2 - RESPONSIVE SECTION

This project is a front-end exercise focused on building a responsive web browser UI using **HTML5 and CSS3**, based on a provided Figma design. The purpose is to reinforce the foundational concepts of responsive design, ensuring the layout adapts well across devices and screen sizes.

## Project Overview

- **Design Source**: 
  - [Figma UI mock-up](https://www.figma.com/design/8p6KclP2GuzXknabZrCjoa/E-commerce-%7C-Website-design-%7C-Bookstore--Community-?node-id=1-414&t=Ct2uJTPblmKxzRRF-0)
- **Tech Stack**:
  - HTML5 ([MDN HTML5 Guide](https://developer.mozilla.org/en-US/docs/Web/Guide/HTML/HTML5))
  - CSS3 ([MDN CSS Reference](https://developer.mozilla.org/en-US/docs/Web/CSS))
- **Goal**:
  - Ensure layout works properly on multiple screen sizes (mobile, tablet, desktop)
  - Use media queries to adapt the layout for popular breakpoints (769px, 1024px)
  - Apply responsive design practices to improve user experience
